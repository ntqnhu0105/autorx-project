### Week 5 Objectives:

* Understand how to build scalable and highly available systems on AWS.
* Learn about load balancing and automatic resource scaling.
* Gain hands-on experience with system monitoring using Amazon CloudWatch.

### Week 5 Achievements:

* Gained a solid understanding of automatic resource scaling with Amazon EC2 Auto Scaling, including:
  * Launch Templates.
  * Auto Scaling Groups.
  * Scaling Policies.
  * Health Checks.

* Learned how to use Elastic Load Balancing to distribute incoming traffic across multiple Amazon EC2 instances, improving system availability and fault tolerance.

* Developed a strong understanding of the core features of Amazon CloudWatch, including:
  * Metrics.
  * Logs.
  * Alarms.
  * Dashboards.
  * Events.

* Learned how to monitor AWS resource performance and configure alarms when system metrics exceed predefined thresholds.

* Understood the key components of Amazon Route 53, including:
  * Hosted Zones.
  * DNS Records.
  * Routing Policies.
  * Domain Name Resolution.

* Successfully configured DNS using Amazon Route 53 and verified domain name resolution.

* Expanded practical AWS knowledge and stayed up to date with the latest cloud technologies by participating in the FCAJ Community Day at the AWS Vietnam Office.

* Acquired the ability to design AWS architectures that meet requirements for scalability, high availability, monitoring, and domain name management.
