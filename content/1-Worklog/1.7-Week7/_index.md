* Learn about AWS services that support infrastructure operations and management.
* Migrate virtual machines to AWS using AWS VM Import/Export.

### Week 7 Achievements:

* Learned how to use AWS VM Import/Export to migrate virtual machines between on-premises environments and AWS.

* Acquired the ability to use AWS Lambda to automate administrative tasks and optimize Amazon EC2 costs.

* Became familiar with Grafana and deployed a basic monitoring solution for AWS infrastructure.

* Learned how to organize and manage AWS resources using:
  * Resource Tags.
  * Resource Groups.
  * Tag-based IAM policies.

* Gained the ability to manage and update multiple Amazon EC2 instances simultaneously using AWS Systems Manager, including:
  * Patch Manager.
  * Run Command.

* Learned how to securely access Amazon EC2 instances using Session Manager without opening SSH ports, while managing session logs and configuring port forwarding.

* Understood the principles of **Infrastructure as Code (IaC)** and successfully deployed AWS infrastructure using AWS CloudFormation templates.

* Strengthened operational, administration, and automation skills by applying AWS Operations best practices across infrastructure management tasks.
