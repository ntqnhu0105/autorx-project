### Week 8 Achievements:

* Developed a solid understanding of AWS security principles and best practices for building secure cloud architectures.

* Successfully deployed AWS IAM Identity Center and learned how centralized identity management works, including:
  * Users.
  * Groups.
  * Permission Sets.
  * Single Sign-On (SSO).

* Learned how to use IAM Permission Boundaries to define the maximum permissions that IAM Users and IAM Roles can receive.

* Understood how to combine IAM Roles with IAM Conditions to build more flexible and secure access control policies.

* Learned how to use AWS Security Hub to:
  * Aggregate security findings.
  * Monitor security compliance.
  * Analyze security findings across AWS resources.

* Successfully deployed and configured AWS WAF to protect web applications against common attacks such as SQL Injection (SQLi) and Cross-Site Scripting (XSS).

* Gained a solid understanding of AWS KMS and data encryption concepts, including:
  * Encryption at Rest.
  * Customer Managed Keys (CMKs).
  * Encryption key management and usage.

* Strengthened the ability to design secure AWS architectures that meet identity management, access control, and data protection requirements.
