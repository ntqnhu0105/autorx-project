### Week 9 Achievements:

* Gained a solid understanding of the AWS Well-Architected Framework, particularly the following pillars:
  * Reliability.
  * Performance Efficiency.
  * Cost Optimization.

* Successfully deployed AWS Backup to protect data and improve system resilience.

* Learned how to connect multiple Amazon VPCs using:
  * VPC Peering.
  * AWS Transit Gateway.

* Successfully deployed Docker-based applications on AWS and gained hands-on experience with CI/CD using AWS CodePipeline.

* Acquired the ability to deploy applications to:
  * Amazon ECS.
  * Amazon EC2 through AWS CodePipeline.

* Understood the role of AWS Storage Gateway and Amazon FSx in integrating and extending storage between on-premises environments and AWS.

* Learned the fundamentals of Data Lake architecture and built a centralized data storage environment on AWS.

* Gained practical knowledge of AWS cost optimization strategies, including:
  * AWS Savings Plans.
  * Reserved Instances.
  * Reserved DB Instances.
  * Right-sizing Amazon EC2 instances.
  * Monitoring and visualizing costs with AWS Cost Management tools.

* Enhanced the ability to design AWS architectures that meet requirements for reliability, performance, and cost efficiency.
