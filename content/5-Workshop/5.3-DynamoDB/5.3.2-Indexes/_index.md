#### Create a Global Secondary Index

To enable searching for vehicles by license plate, we will create a **Global Secondary Index (GSI)**.

Select the **SmartVehicle_Vehicles** table.

![table](/images/5-Workshop/5.3-DynamoDB/5.3.2-Indexes/5.3.2_1.jpg)

Navigate to the **Indexes** tab.

Choose **Create index**.

![table](/images/5-Workshop/5.3-DynamoDB/5.3.2-Indexes/5.3.2_2.jpg)

Configure the index as shown below.

![table](/images/5-Workshop/5.3-DynamoDB/5.3.2-Indexes/5.3.2_3.jpg)

Then choose **Create index** and wait a few moments for the index status to become **Active**.

![table](/images/5-Workshop/5.3-DynamoDB/5.3.2-Indexes/5.3.2_4.jpg)
