# Dọn dẹp tài nguyên (Cleanup)

{{% notice warning %}}
⚠️ **Quan trọng:** Sau khi hoàn thành Workshop, bạn cần xóa tất cả tài nguyên đã tạo để tránh phát sinh chi phí không mong muốn trên tài khoản AWS của mình.
{{% /notice %}}

#### Các bước dọn dẹp:
1. Xóa các Things và Certificates trên **AWS IoT Core**.
2. Xóa các Rules trên IoT Core.
3. Xóa các bảng trên **Amazon DynamoDB**.
4. Xóa các hàm trong **AWS Lambda**.
5. Xóa API trên **Amazon API Gateway**.
6. Xóa Alarm và Log Groups trên **Amazon CloudWatch**.
7. Xóa Topic trên **Amazon SNS**.
