# Resource Cleanup

{{% notice warning %}}
⚠️ **Important:** After completing the Workshop, you must delete all created resources to avoid unexpected charges on your AWS account.
{{% /notice %}}

#### Cleanup steps:
1. Delete Things and Certificates on **AWS IoT Core**.
2. Delete Rules on IoT Core.
3. Delete tables on **Amazon DynamoDB**.
4. Delete functions in **AWS Lambda**.
5. Delete APIs on **Amazon API Gateway**.
6. Delete Alarms and Log Groups on **Amazon CloudWatch**.
7. Delete Topics on **Amazon SNS**.
