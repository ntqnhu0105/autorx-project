# Building a Smart Vehicle Management System on AWS

#### Overview

In this workshop, we will practice building a **Smart Vehicle Management** system using Serverless and IoT services on AWS.

The system will collect operational data from vehicles (speed, engine temperature, mileage), process and store it on the Cloud, and automatically send alerts when an anomaly occurs or maintenance is due.
